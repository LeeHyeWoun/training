> <p><strong>File_encryption_program_using_OpenSSL.cpp</strong></p>
> 수업 : 현대암호실습및응용<br>
> 날짜 : 2020.07.05. (금) 15주차<br>
> 과제 : 암호 기법을 적용한 프로그램 만들기<br>
> 기능 :<br>
> * 원하는 경로의 설정과 디렉토리 내의 파일 목록을 확인하고 암호화 및 복호화를 원하는 파일 선택 가능합니다.<br>
> * openssl의 MD5 암호화 함수를 사용해 선택한 파일(원본)을 읽어와 암호화된 HASH 문자열을 출력합니다. <br>
> * openssl의 AES 암호화 함수를 사용해 선택한 파일(원본)을 읽어와 암호화된 파일(확장자는 enc로 지정)을 생성합니다.<br>
> * openssl의 AES 복호화 함수를 사용해 선택한 파일(원본)을 읽어와 암호화된 파일(확장자는 enc로 지정)을 생성합니다.<br>
